Implementação de VPC Personalizada com EC2 para Execução de Servidor Web

Neste projeto, criei um ambiente de nuvem privada virtual (VPC) personalizado utilizando o Amazon Web Services (AWS). A configuração envolveu a criação de sub-redes, grupos de segurança e o lançamento de uma instância do Amazon EC2 dentro da VPC. A instância foi configurada para executar um servidor web.

Arquitetura

Introdução ao Amazon EC2

Projeto focado na utilização do Amazon Elastic Compute Cloud (EC2) para gerenciar e escalar servidores na nuvem. Através de uma interface simples, configurei uma instância de servidor com rapidez e flexibilidade. O projeto incluiu a execução de um servidor web com proteção contra encerramento ativada, monitoramento da instância, ajustes de grupos de segurança para permitir acesso HTTP, e o redimensionamento da capacidade conforme necessário. Também explorei limites do EC2, testei a proteção contra encerramento e realizei o encerramento seguro das instâncias.

Página de evento agendado - Copa do Mundo

Landing page para a Copa do Mundo com um contador regressivo em JavaScript. Estruturei o projeto com Parcel e organizei os elementos de design, como cores e tipografia. Apliquei gradientes, imagens responsivas e estilizei o cabeçalho e o layout para garantir a responsividade. Usei variáveis CSS e animações externas para um design atraente, e implementei a contagem regressiva com getTime, setInterval, e clearInterval para criar uma experiência interativa.
